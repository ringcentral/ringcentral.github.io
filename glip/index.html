Glip APIs & SDKs - Preview Release

Glip REST APIs are part of the RingCentral Platform REST APIs and use the same Developer Portal and OAuth Authorization. As such, the core RingCentral SDKs will work for Glip REST APIs. The information provided below is specific additional information and resources for the Glip APIs.

Please reach out to us at devsupport@ringcentral.com if you wish to get access to these APIs during the preview release stage.
